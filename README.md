
# Networking Practice Labs

This repository contains a collection of networking labs created for continuous hands-on practice and skill development. The focus is on building practical experience in routing, troubleshooting, and real-world network scenarios.

## Purpose

The goal of this repository is to practice and reinforce networking concepts, build practical experience beyond theory, simulate real-world network environments, improve troubleshooting and problem-solving skills, and prepare for roles such as NOC Analyst and Network Engineer.

## What You Will Find

Structured lab scenarios, routing-based configurations, troubleshooting exercises, progressive difficulty levels, and realistic network designs.

## Lab Structure

Each lab typically includes:

- questions.md – tasks and objectives  
- answers.md – solutions and configurations  

## Technologies Covered

IPv4 addressing and subnetting, routing protocols such as RIP and OSPF, route redistribution, static and default routing, network troubleshooting, and basic security and device hardening.

## Environment

These labs are built and tested using EVE-NG and its associated devices.

## Approach

This is a hands-on learning repository where labs are designed to encourage critical thinking rather than step-by-step copying. Some tasks are intentionally structured to require troubleshooting, with a focus on understanding concepts and behavior.

## Notes

These labs are for practice and learning purposes. Configurations may vary depending on topology and approach, and updates will be made over time as the repository evolves.
